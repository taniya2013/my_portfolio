🎓 University Dashboard

Project Overview

This interactive Power BI dashboard provides insights into university student performance, academic results and departmental analysis. It helps users monitor student records through KPIs, charts and filters.

Dashboard Features

  • 📊 Count of Department by Student Name
  • 🎯 Total Pass and Expected Pass by Subject
  • 📈 Count of Student ID by Year
  • 🍩 Count of City by State
  • 📉 Count of Grade by Semester
  • 👨‍🎓 Count of Gender by Year and City
  • 🏫 Count of Student ID by Department
